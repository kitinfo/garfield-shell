This application handles interaction with the snack subsystem of Garfield,
such as finding and buying snacks, as well as displaying some fun statistics
and information.

garfield-snack can be built on Windows as well as Linux-based systems.
The provided makefile uses tcc as the standard compiler, gcc can be invoked
by using the make target 'gcc' or, like any other compiler, by typing the
commands yourself (it ain't exactly rocket science).
